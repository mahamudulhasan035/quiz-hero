JavaScript Array methods: map() Vs forEach() Vs filter() Vs find()

1. map()

The array map() method works with each element of an array and returns a new array.


2. forEach()

The array forEach() method works with each element of an array but doesn't returns a new array.


3. filter()

The array filter method works with those array elements which fulfil single or multiple conditions. The filter method returns a new array. If no array elements fulfilled the conditions, then it will return an empty array.


4. find()

The array finds method searches for a specific array element with a condition. If the element is found within the array, then it will return the element. If no element fulfilled the condition, then it will return undefined. If multiple elements fulfil the condition, the find method will return the first matched element.

What is JavaScript Event loop? How does it works?

In JavaScript, when some code needs to be executed, it will go to the call stack. Call stack works as LIFO method (Last In First Out). If the code's default behavior is synchronous, then it will execute and pop up from the call stack. But if its behavior is asynchronous, then it will go to the Web API / Browser API and performs its task there. Such as (setTimeout, setInterval, fetch(), DOM Event, etc). When the assigned task is complete, these codes will go to the Queue which works as a FIFO method (First In First Out) and wait there. Now the Event loop will work. The event loop will check if the call stack is empty or not. If it is, then will send the code from the queue to call the stack one by one.

What does quizTimer() function do?

The quizTimer() function takes a parameter which is a boolean value. If it's true, then it will stop the setInterval() function. Inside the function calculates how much minutes and seconds a user takes to complete all the quizzes. Depending on the second number (even or odd) it changes the color of the time.

What does chooseQuiz() function do?

The chooseQuiz() function takes two parameters which are index (the index of the quizData array), and givenAns (the answer the user has chosen). First, it checks if the answer is already in the array or not. If it does, then replace it with the new answer, otherwise, add it to the array.
